## 🏗️ TECHNICAL STACK

### Frontend
- **Framework**: Next.js 14 (React 18)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Forms**: Formik + Yup
- **Language**: TypeScript

### Integrations
- **Maps**: Google Maps API
- **Reviews**: Google Reviews API
- **Analytics**: Google Analytics 4
- **Hosting**: Vercel
